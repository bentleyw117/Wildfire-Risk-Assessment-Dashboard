# Installation

## Stable release

To install Wildfire Risk Assesment Dashboard, run this command in your terminal:

```sh
uv add wildfire-risk-dashboard
```

Or if you prefer to use `pip`:

```sh
pip install wildfire-risk-dashboard
```

## From source

The source files for Wildfire Risk Assesment Dashboard can be downloaded from the [Github repo](https://github.com/bentleyw117/wildfire_risk_dashboard).

You can either clone the public repository:

```sh
git clone git://github.com/bentleyw117/wildfire_risk_dashboard
```

Or download the [tarball](https://github.com/bentleyw117/wildfire_risk_dashboard/tarball/main):

```sh
curl -OJL https://github.com/bentleyw117/wildfire_risk_dashboard/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd wildfire_risk_dashboard
uv pip install .
```
